Column Definition
Screenshot Snapshot of the website provided by Business Process Incubator
Tool Suite Name of the tool suite to which a tool belongs

BPMN 2.0 Tool supports BPMN version 2.0 or higher
MIWG Member Vendor is member of the BPMN MIWG, i.e. at least subscribed to the mailing list
Test Results submitted Test results have been submitted for the tool either by users or the vendor
Import Tool has a feature to import BPMN 2.0 XML files
Export Tool has a feature to export BPMN 2.0 XML files
Roundtrip Tool can both import and export BPMN 2.0 XML files
BPMN Element Coverage Percentage of BPMN elements that a tool supports (metric calculated by BPMN MIWG Test Automation Tools)
Preserves Layout Tool preserves positions of BPMN elements during a roundtrip (metric calculated by BPMN MIWG Test Automation Tools)
Open Issues Issues identified by the BPMN MIWG that are still open
Closed Issues Issues identified by the BPMN MIWG that are already fixed by the vendor
Participates in Demos Tool participated in one of the public BPMN MIWG demos
Links Further information provided by Business Process Incubator
